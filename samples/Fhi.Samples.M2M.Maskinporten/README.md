# Maskinporten sample
Maskinporten OAuth 2.0 token for machine-to-machine communication. Maskinporten does not use [OAuth 2 client credentials grant](https://datatracker.ietf.org/doc/html/rfc6749#section-1.3.4) 
but uses an OAuth 2 extension grant type, specified in [RFC 7523 OAuth 2.0 extension profile](https://datatracker.ietf.org/doc/html/rfc7523)

- [Resource Indicators for OAuth 2.0](https://datatracker.ietf.org/doc/html/rfc8707)

This sample demonstrates how to obtain and use a Maskinporten token in a .NET application using nuget package from Altinn, [altinn-apiclient-maskinporten](https://github.com/Altinn/altinn-apiclient-maskinporten/tree/main)
